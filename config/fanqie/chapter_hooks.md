# 番茄小说章末钩子模板
# 目标：引导读者在章说讨论，留下追读悬念

## 核心理念

番茄的"章说"是读者互动核心。好的章末钩子让读者：
1. 忍不住想说点什么（讨论钩子）
2. 忍不住想看下一章（追读钩子）

```
章末钩子 ≠ 简单断章
章末钩子 = 制造"信息差"或"情感冲突"
```

## 钩子分类（7种核心类型）

### 1. 悬念式钩子（最常用）
```
留下未解之谜，读者必须看下去才能知道答案
```

**模板**：
- "他话说到一半，突然停住了——门外，有脚步声。"
- "师父的脸色变了。'你不该来这里。'他说，'他回来了。'"
- "张凡低头看着手中的玉简，上面只有一个字——'逃'。"
- "就在她伸手的瞬间，天空裂开了一道缝隙。"

**要点**：
- 不要透露具体是什么
- 制造紧张感或好奇
- 短促有力

---

### 2. 反转型钩子
```
颠覆读者预期，制造认知冲击
```

**模板**：
- "张凡一剑刺穿了黑影——然后他看清了那张脸，那是他自己的脸。"
- "所有人都以为他会死。但他笑了，因为一切都在计划中。"
- "原来，真正的敌人从来不是魔族——而是他身边最信任的人。"
- "她以为自己在利用他，直到看见他留下的那封信。"

**要点**：
- 前面要有足够的铺垫
- 反转要合理，不能为了反转而反转
- 反转后要有新的悬念

---

### 3. 冲突式钩子
```
制造强烈的矛盾冲突，让读者想看如何解决
```

**模板**：
- "张凡跪在父亲灵前，发誓要为张家报仇——但他不知道，凶手就站在他身后。"
- "师父要他杀的那个人，是他失散多年的亲弟弟。"
- "她握紧了手中的剑，指向她曾经最爱的人。"

**要点**：
- 冲突必须是核心矛盾
- 双方都有正当理由
- 没有简单的是非对错

---

### 4. 情感式钩子
```
留下情感上的牵挂，让读者关心角色命运
```

**模板**：
- "她望着他的背影消失在夜色中，嘴唇动了动，却什么都没说出来。"
- "张凡不知道的是，这是他最后一次见到师父的笑容。"
- "他不知道，在地底的深处，有一双眼睛正望着他——带着千年的等待。"

**要点**：
- 不要过度煽情
- 用细节传递情感
- 给读者留下想象空间

---

### 5. 抉择式钩子
```
让读者想知道角色会怎么选
```

**模板**：
- "张凡面前有两条路：一条通向真相，一条通向——死亡。"
- "她必须在宗门和亲人之间做出选择。而时间，只有一天。"
- "系统的声音响起：'选择吧。拯救一个人，放弃另一个人。'"

**要点**：
- 选择必须两难
- 两个选项都有代价
- 读者会有自己的倾向

---

### 6. 危机式钩子
```
制造紧迫感，让读者想看如何化解
```

**模板**：
- "结界正在崩溃。而他还有三分钟。"
- "体内的灵力正在流失，而敌人的脚步声越来越近。"
- "倒计时开始：10、9、8——"

**要点**：
- 明确的时间压力
- 危机要具体
- 倒计时要有数字感

---

### 7. 信息差钩子
```
让读者知道一些角色不知道的事，制造张力
```

**模板**：
- "只有读者知道，那封信里的毒，三天后才会发作。"
- "他们不知道的是，在他们身后，还有一双眼睛在看着这一切。"
- "张凡以为自己在追敌人——但读者知道，真正的危险在后方。"

**要点**：
- 制造"上帝视角"张力
- 读者比角色知道更多
- 信息差要有戏剧性用途

---

## 钩子组合公式

好的章末往往是**组合钩子**：

```
悬念 + 冲突 = "他发现了真相——但这个真相会让他失去一切"
情感 + 反转 = "她为他挡下了那一剑——然后才意识到她爱错了人"
危机 + 抉择 = "结界的裂缝越来越大，而他必须在使命和爱之间选择"
```

---

## 章末禁句（绝对不要写）

```
❌ "欲知后事如何，请看下回分解"（太老套）
❌ "这一章结束了"（废话）
❌ "张凡决定..."（没有冲突的平淡过渡）
❌ "三天后..."（没有悬念的时间跳跃）
❌ 大段总结性文字（把读者当傻子）
❌ 没有钩子，直接断在"然后"
```

---

## 每章钩子自检清单

```
□ 有没有留下未解之谜？
□ 读者看完这章后最想知道什么？
□ 这个悬念能否用一句话概括？
□ 钩子是否在章节最后1-2句？
□ 有没有制造信息差或情感冲突？
□ 读者会不会想去章说讨论？
```

---

## 不同节奏点的钩子选择

| 章节位置 | 推荐钩子类型 | 说明 |
|---------|------------|------|
| 第1-3章 | 悬念式/危机式 | 建立期待，读者要决定是否追读 |
| 第4-10章 | 冲突式/情感式 | 建立角色情感连接 |
| 第10-30章 | 反转型/抉择式 | 剧情展开，需要更多张力 |
| 30章+ | 组合钩子 | 综合运用，维持高张力 |

---

## 实战范例

### 弱钩子（不要这样写）
```
张凡回到家中，想了想今天发生的事，决定明天继续修炼。
（完）
```

### 强钩子（这样写）
```
张凡回到家中，想了想今天发生的事。
然后他发现了师父留下的那封信。
信上只有一句话："不要相信任何人，包括我。"
他的手开始颤抖——不是因为害怕，而是因为信纸上的血迹还没干。
（章说：师父到底怎么了？？？）
```
