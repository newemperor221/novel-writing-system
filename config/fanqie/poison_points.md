# 番茄小说毒点清单
# 踩了这些，轻则无人问津，重则直接弃书

## 第一章毒点（读者决定是否追读的关键）

### 必死的开局模式
1. **开局送女** — 第一章就出现倒贴女性角色
2. **孤儿开局无代入** — 主角父母双亡但没有任何情感铺垫
3. **系统/签到全文堆** — 开局就秀系统，读者知道后面是挂机文
4. **战力崩坏前兆** — 第一章就出现秒杀/越级杀怪
5. **水字数型环境描写** — 大段与剧情无关的风景/设定介绍
6. **人设崩塌** — 主角前脚说"绝不求人"，后脚跪舔

### 前三章核心检查点
```
□ 没有期待点（钩子太弱，读者不知道在期待什么）
□ 主角没有明确目标（读者不知道主角要干嘛）
□ 没有冲突/矛盾（太平淡，无聊）
□ 信息过载（一次性抛太多设定）
□ 节奏拖沓（该推进的没推进）
```

## 中期毒点（读者决定是否继续追读）

### 套路陈旧
1. **退婚流** — 除非写法创新，否则已经被写烂
2. **废柴逆袭** — 开局被欺负，没有新意
3. **升级就是数字变化** — 境界变了但看不出变强
4. **战斗就是技能名堆砌** — 没有招式描写，只有"大招"名称
5. **所有反派都一个模子** — 阴险、好色、愚蠢三件套

### 人设问题
1. **女主工具人** — 女性角色只为推动剧情存在
2. **配角纸片人** — 除了主角都是背景板
3. **主角圣母** — 敌人杀他全家，他放过敌人
4. **主角龙傲天** — 开局就天下无敌，无敌文
5. **反派强行降智** — 为了让主角赢，反派变蠢

### 节奏问题
1. **重复循环** — 升级→打脸→升级→打脸，循环到腻
2. **支线灌水** — 与主线无关的支线占大量篇幅
3. **回忆杀滥用** —动不动就大段回忆
4. **对话水字数** — 角色说废话，不推进剧情
5. **高潮被腰斩** — 最精彩的地方没写完就跳过

## 高级毒点（老读者一眼就弃）

### 世界观崩塌
1. **境界通货膨胀** — 从炼气到飞升只用50章
2. **规则被吃** — 前面立的规则后面不遵守
3. **时间线混乱** — 前面说"三天后"后面变"三个月后"
4. **地图随便换** — 没有地理概念，随意穿越

### 情感处理
1. **强行煽情** — 不到位的情感爆发让读者尴尬
2. **感情线工业糖精** — 无脑甜，没有发展过程
3. **死亡没有重量** — 角色死得太随意
4. **反派洗白太轻易** — 洗白没有足够铺垫

### 文笔问题
1. **AI味太重** — 因果链、机械节奏、无聊的形容词
2. **对话书面化** — 角色说话像念稿，不像活人
3. **动作描写缺失** — 只有"说""看"，没有动作
4. **心理描写泛滥** — 大段内心独白代替剧情推进

## 毒点优先级（必须优先避开的）

### CRITICAL — 直接弃书
- 开局送女
- 圣母主角
- AI味（因果链、机械节奏）

### HIGH — 严重影响追读
- 战力崩坏
- 反派降智
- 循环套路无新意

### MEDIUM — 逐渐流失读者
- 支线灌水
- 回忆杀滥用
- 女主工具人

### LOW — 降低阅读体验
- 配角纸片人
- 战斗描写流水账
- 对话水字数
