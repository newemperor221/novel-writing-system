# 番茄小说节奏参数
# 目标：每章3000字，打出最佳追读效果

## 番茄阅读行为分析

```
番茄读者特征：
- 碎片时间阅读（通勤、午休、睡前）
- 快速扫描，2-3分钟看一章
- 追读率高但耐心低
- 章说是重要互动和决策点
```

## 字数分配模板（3000字/章为例）

### 基础节奏：5-3-2-2-1 法则

```
总字数的50% = 1500字（铺垫/建置）
总字数的30% = 900字（冲突/对抗）
总字数的20% = 600字（高潮/解决）

具体到3000字：
├── 铺垫（1500字）
│   ├── 开篇钩子（200字）
│   ├── 场景建立（300字）
│   ├── 角色状态（300字）
│   ├── 目标/动机呈现（400字）
│   └── 第一个小冲突（300字）
├── 冲突（900字）
│   ├── 冲突升级（400字）
│   ├── 意外转折（300字）
│   └── 关键抉择（200字）
├── 高潮（600字）
│   ├── 最高潮时刻（300字）
│   └── 结果呈现（300字）
└── 钩子（100字）
    └── 章末悬念（100字）
```

## 章节结构模板

### 1. 开篇钩子（前200字）

**目标**：3秒内抓住读者，否则读者会切走

**公式**：
```
[悬念/危机/反常] + [情绪] + [期待]
```

**范例**：
```
张凡的剑刺入敌人胸口的那一瞬间，他听到了一个声音——
是他师父的声音。
"杀错人了。"
（开篇就是危机+悬念）
```

**禁忌**：
```
❌ 大段环境描写
❌ 设定介绍
❌ 前情回顾
❌ 慢热铺垫
```

### 2. 场景建立（300字）

**目标**：让读者知道"在哪里、谁在场"

**要点**：
- 1-2个感官细节（不是大段描写）
- 快速点名地点和人物状态
- 不要超过300字

**范例**：
```
宗门后山，夜色沉沉。
张凡单膝跪地，胸口起伏不定。刚才那一剑耗尽了他最后的灵力。
月光下，他的脸色苍白如纸。
```

### 3. 冲突升级（400字）

**目标**：制造第一个小高潮

**公式**：
```
冲突发起 → 对方反应 → 张力加大 → 意外出现
```

### 4. 关键转折（300字）

**目标**：打破读者预期

**公式**：
```
读者以为 → 实际是 → 新问题出现
```

### 5. 章末钩子（最后100字）

**目标**：让读者想看下一章

见 `chapter_hooks.md`

## 不同章节类型的节奏调整

### 战斗章节（打脸/升级）

```
铺垫：20%（建立敌人有多强）
冲突：50%（战斗过程，一波三折）
高潮：25%（逆转或突破）
钩子：5%（留下新悬念）
```

### 情感章节（师徒/爱情/友情）

```
氛围：30%（场景+情绪铺垫）
互动：40%（角色互动，对话为主）
转折：20%（情感高潮或破裂）
余韵：10%（章末情感钩子）
```

### 剧情推进章节（揭秘密/世界观）

```
铺垫：25%（制造期待）
揭秘：50%（信息释放，层层剥开）
冲击：20%（揭秘后的反应）
钩子：5%（新的谜团）
```

### 日常章节（蓄力/伏笔）

```
场景：30%（让读者喘口气）
小事：30%（建立日常，但要有信息量）
暗流：30%（表面平静实际有伏笔）
悬念：10%（微妙的钩子）
```

## 番茄特有效率指标

### 每500字的"钩子密度"

番茄读者平均阅读速度：1000字/分钟
所以3000字 = 3分钟阅读时间

```
3分钟内，读者需要：
- 1个明确的期待点
- 1个情感连接点
- 1个讨论点（章说素材）
```

**检查**：
```
□ 每500字有没有一个"为什么？"
□ 每1000字有没有一个"然后呢？"
□ 每章有没有一个让读者想发章说的点？
□ 每3章有没有一个"我猜到了！"的伏笔回收？
```

### 追读率关键节点

```
第1章：留存率决定是否上推荐
第3章：追读率第一个窗口
第10章：第二个追读窗口
第30章：长篇读者筛选
第50章+: 核心粉丝养成
```

## 节奏自检清单

```
□ 开篇200字有没有钩子？
□ 3分钟内有没有一个小高潮？
□ 有没有至少一个意外转折？
□ 章末有没有让读者想看下一章的悬念？
□ 有没有超过200字的纯描述段落？
□ 对话有没有推进剧情（而不是水字数）？
□ 有没有"信息差"让读者比角色知道更多？
□ 章说讨论点在哪里？
```

## 常见节奏问题

### 问题1：太平淡
**表现**：章节看完了，读者什么都不记得
**解决**：增加冲突密度，每500字一个"为什么"

### 问题2：节奏太快
**表现**：情节推进了但读者没代入
**解决**：在高潮前加情绪铺垫

### 问题3：水字数
**表现**：同样的场景/对话写了又写
**解决**：每段话都要推进剧情或揭示角色

### 问题4：章章打脸
**表现**：套路重复，读者疲劳
**解决**：三章一个小循环，十章一个大循环
