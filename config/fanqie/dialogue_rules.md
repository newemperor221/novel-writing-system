# 中文网文对话守则
# 目标：写出像人说话的对话，而不是念稿

## 核心原则

```
对话是角色的声音，不是作者的声音
好的对话：听得到语气，看得到表情，知道是谁在说
```

## 1. 对话标签（可用/禁用）

### 可用标签（简短有力）
```
说、问、答、喊、吼、叹、笑、怒、哼、骂
```

### 禁用标签（太书面/太水）
```
❌ 轻声说道 ❌ 低声说 ❌ 缓缓说道 ❌ 慢慢回答
❌ 叹了口气说 ❌ 微笑着说我 ❌ 意味深长地说
❌ 点了点头说 ❌ 皱起眉头说
```

### 正确示范
```
✗ 弱："张凡轻声说道："我一定会回来的。""
✓ 强："张凡："我一定会回来。"他的声音很轻，但很坚定。

✗ 弱："王浩低声说："你完了。""
✓ 强："王浩："你完了。"他抬起下巴，俯视着张凡。

✗ 弱："师父微笑着摸了摸他的头："好孩子。""
✓ 强："师父摸了摸他的头："好孩子。"眼里有笑意，但笑意不达眼底。
```

## 2. 对话功能检测

每段对话必须满足至少一条：

| 功能 | 检查 |
|------|------|
| 推进剧情 | 这句话让事情往前走了吗？ |
| 揭示性格 | 这话像谁说的？换个角色说不说同样的话？ |
| 建立关系 | 这话是拉近还是推开？ |
| 制造冲突 | 这话有没有让矛盾升级？ |
| 埋伏笔 | 这话有没有暗示后续？ |

**如果一条都不满足 = 水对话 = 删掉**

## 3. 对话句式模板

### 反问句（强势/冲突）
```
"你以为我会信？"
"你觉得可能吗？"
"谁告诉你……"
"我凭什么？"
```

### 威胁句（压迫感）
```
"最后一次机会。"
"别让我说第二遍。"
"你可以试试。"
"后果自负。"
```

### 疑惑句（好奇/探究）
```
"什么意思？"
"你想说什么？"
"……然后呢？"
"等等，你怎么知道？"
```

### 情感句（真实/有力）
```
"别走。"
"我恨你。"
"你会回来的，对吧？"
"我不后悔。"
```

## 4. 对话节奏控制

### 短长交替
```
短句建立节奏，长句增加信息：
张凡没说话。
王浩笑了："我知道你不服。但不服没用。"
张凡的拳头握紧。他看着王浩，一字一句："我会让你后悔。"
```

### 动作插入
```
不用每次都用"他说"
可以：
- 用动作打断
- 用反应打断
- 用沉默打断

范例：
"你以为你赢定了？"张凡站起身，椅子倒了。
王浩没动。
"我不会让你得逞。"
王浩笑了："你以为你有选择？"
```

### 对话精简
```
✗ 水："师兄，你说的那个地方，我们真的要去吗？那里是不是很危险？
       我有点担心，如果我们遇到敌人怎么办？你能保护我吗？"

✓ 精："师兄，那地方……有把握吗？"
```

## 5. 常见问题

### 问题1：对话像念稿
```
✗ 弱："张凡说道："今天天气真好，我们去修炼吧。"
       王浩说道："好啊，我们去后山修炼吧。"
       张凡说道："是的，我们走吧。"

✓ 强：張凡踢了踢王浩的凳子："走，后山。"
       王浩头也不抬："又去？"
       "不去拉倒。"
       王浩叹了口气，站起来："行行行，走。"
```

### 问题2：对话没有角色区分
```
✗ 弱：所有人说话风格都一样

✓ 强：
- 主角张凡：直接、简洁、有时带点痞气
- 师父：沉稳、话不多、每句都有分量
- 女配：小女儿态、会撒娇、偶尔倔强
- 反派：嚣张、爱威胁、说话带刺

检测：把名字遮掉，能不能从话认出是谁说的？
```

### 问题3：对话推进无力
```
✗ 弱："你来了。" "我来了。" "你吃了吗？" "吃了。"

✓ 强："你来了。"
"我来了。"
沉默。
张凡先开口："东西带了？"
"带了。"王浩从怀里掏出玉简，"但我有个条件。"
```

### 问题4：连续对话太长
```
超过3轮连续对话要打断

✗ 弱：连续10轮对话没动作

✓ 强：
"你说的那个秘密——"张凡刚要开口。
王浩抬手打断他："闭嘴。"
"……"
"我知道你要问什么。"王浩的眼神变了，"但你不会想知道答案。"
```

## 6. 对话字数控制

```
单次对话：10-30字为宜
超过50字要检查：有没有可以删的？
对话+动作描写：可以适当延长

快速对话（紧张场景）：
"枪！"
张凡侧身，躲过。
"还是慢了。"
他低头，胸口多了一道血痕。

慢速对话（情感场景）：
"……"
"你倒是说话啊。"
她还是不开口。
他叹了口气，转身要走。
"别走。"
就两个字，但声音在抖。
```

## 7. 章节对话密度

```
3000字章节建议：
对话总字数：600-900字（20-30%）
对话轮次：15-25轮
每轮平均：25-40字

检查：
□ 有没有连续3轮以上无动作的纯对话？
□ 有没有超过50字的单次对话（除非特别需要）？
□ 对话有没有推进剧情？
□ 不同角色的对话风格能区分开吗？
```
