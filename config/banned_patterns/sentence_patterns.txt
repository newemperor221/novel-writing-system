# AI 句式模式 — 检测并避免

# 模式1: AI 因果链 "A 感到 B，因为 C，因此 D"
# 匹配: 某人说"因为...所以/因此..."
^[^。\n]*感到[^。\n]*，因为[^。\n]*，因此

# 模式2: "只见" 开头过度使用
# 限制: 每500字最多1次
^只见

# 模式3: 连续3+句以相同主语开头
# 匹配: 3行或以上连续以"他"/"她"/"它"开头
^(他|她|它)[^。\n。]*\n(他|她|它)[^。\n。]*\n(他|她|它)

# 模式4: "被...所..." 被动句过度使用
# 限制: 每300字最多1次
^[^。\n。]*被[^。\n。]*所[^。\n。]*

# 模式5: "开始...开始" 重复
开始.开始

# 模式6: 句末使用"了起来"过度
了起来。.*

# 模式7: 连续短句（AI倾向短句堆砌）
^[^。]{1,5}\。[^。]{1,5}\。[^。]{1,5}\。

# 模式8: "而" 字句过度开头的段落
^而

# 模式9: 段落完全由对话组成（无动作/描述）
^「[^」]*」\n^「[^」]*」\n^「[^」]*」
